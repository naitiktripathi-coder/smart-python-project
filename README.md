# 🎮 ROCK PAPER SCISSOR GAME 🎮
Rock Paper Scissor Game
A simple command-line Rock Paper Scissor game built using Python.
This project is designed for learning Python basics, logic building, and entertainment.
Description
This program allows a user to play the classic Rock, Paper, Scissor game against the computer.
It gives 10 chances to the user and computer.
The computer randomly selects its choice, and the program determines the winner based on the rules of the game.
The game also uses colored terminal output to make the experience more interactive and visually appealing.
Features
• Play Rock, Paper, Scissor against the computer
• Computer makes random choices using Python’s random module
• Tracks user score and computer score
• Detects draw, win, and lose conditions
• Uses the Colorama library to display colorful terminal text
• Option to send a celebration emoji 🎉 to the computer when it wins
• User can press Q to quit the game anytime
Technologies Used
• Python
• Random Module
• Colorama Library
How to Run
Install Python on your system
Install the Colorama library
Copy code

pip install colorama
Run the program
Copy code

python game.py
Type rock, paper, or scissor to play
Type Q to quit the game
Example Gameplay
Copy code

Type ROCK, PAPER, SCISSOR and Q for quit:
rock
Computer Chose: paper
Computer Win 🤖
Your Score: 0
Computer Score: 1

# 🚀 Smart Search Engine (App Search Hub)

A simple and fun Python-based app that allows users to search directly on popular platforms like YouTube, Facebook, Instagram, GitHub, and Snapchat from the terminal.

---

## 🌟 Features

- 🔍 Search directly on:
  - ▶️ YouTube
  - 📘 Facebook
  - 📸 Instagram
  - 🐙 GitHub
  - 👻 Snapchat
- 🎨 Colorful terminal UI using `colorama`
- 🔁 Infinite loop (search again and again)
- ❌ Error handling for invalid input
- 🚪 Exit option
